# Willfy Mofa Pet

So, it's just a pet of twitch streamer Willfy Mofa. She can walk, jump, clrawling (climbing). You can her dragging, but she doesn't like that.

## Created with
I just wanted to create desk pet. This small project was created with:

[GCC](https://gcc.gnu.org/), [GoDot](https://godotengine.org/), [Template for GDExtension C++](https://github.com/godotengine/godot-cpp-template), [godot-cpp](https://github.com/godotengine/godot-cpp/tree/4.5), [Chatty_desktop_pet](https://github.com/ExtraNick/Chatty_desktop_pet) and [this tutorial](https://github.com/geegaz/Multiple-Windows-tutorial). There some others sources but this is a core.
